If / Else Statements

Make decisions in your JavaScript code using conditional statements

âœ… Basic if Statement

The if statement runs a block of code only if a condition is true.

let age = 20;

if (age >= 18) {
  console.log("You are an adult.");
}

âœ… if / else Statement

Add an else block to run a different code when the condition is false.

let age = 16;

if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

âœ… else if Statement

You can add multiple conditions using else if.

let score = 75;

if (score >= 90) {
  console.log("Grade A");
} else if (score >= 80) {
  console.log("Grade B");
} else if (score >= 70) {
  console.log("Grade C");
} else {
  console.log("Fail");
}
