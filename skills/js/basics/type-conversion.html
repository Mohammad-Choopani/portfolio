Type Conversion

Learn how JavaScript handles automatic and manual type conversion

üîÑ Implicit Conversion (Type Coercion)

JavaScript tries to automatically convert types in some operations:
  • '5' + 2 ‚Üí "52" (number becomes string)
  • '5' - 2 ‚Üí 3 (string becomes number)
  • true + 1 ‚Üí 2 (true becomes 1)
console.log('5' + 2); // "52"
console.log('5' - 2); // 3
console.log(true + 1); // 2

üß™ Explicit Conversion

Use functions like Number(), String(), and Boolean():
  • Number("123") ‚Üí 123
  • String(456) ‚Üí "456"
  • Boolean(0) ‚Üí false
let num = Number("42");
let str = String(123);
let bool = Boolean("");
