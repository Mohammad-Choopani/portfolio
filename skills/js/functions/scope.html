Scope in JavaScript

Learn how variable scope works in JavaScript and avoid common pitfalls!

Scope defines **where a variable is accessible** in your code. JavaScript has three main types of scope:
  • Global Scope: Available anywhere in your script.
  • Function Scope: Available only inside a function.
  • Block Scope: Available inside {} blocks (only with let and const).
let x = 10; // Global

function test() {
  let y = 20; // Function Scope
  if (true) {
    let z = 30; // Block Scope
    console.log(x, y, z); // ‚úÖ
  }
  console.log(z); // ‚ùå ReferenceError
}

test();
