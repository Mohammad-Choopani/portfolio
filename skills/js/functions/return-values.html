Return Values

Understand how to return data from functions and use them in your code

üîÅ What is return?

The return statement ends a function and sends a value back to the caller.

  • Once return is executed, the function stops.
  • The returned value can be stored in a variable or used directly.
  • If no return is written, the function returns undefined.
function sum(a, b) {
  return a + b;
}
let total = sum(5, 10);
console.log(total); // Output: 15
