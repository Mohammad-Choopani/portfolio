Arrow functions are a shorter way to write functions using the => syntax. They are especially useful in callbacks, array methods, and one-liners.
  • No function keyword
  • this behaves differently (lexical binding)
  • No arguments object
// Traditional function
function add(a, b) {
  return a + b;
}

// Arrow function
const addArrow = (a, b) => a + b;

console.log(addArrow(5, 3)); // 8
