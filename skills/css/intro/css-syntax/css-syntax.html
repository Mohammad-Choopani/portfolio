CSS Syntax

In this lesson, you'll learn the correct structure of a CSS rule and how to write it properly.

What is CSS Syntax?

Each CSS rule consists of:

  • Selector ‚Äì selects the HTML element
  • Property ‚Äì the style you want to change

Structure

  property: value;

Example

  color: blue;
  font-size: 24px;
}

Multiple Properties

  color: green;
  font-family: Arial, sans-serif;
  line-height: 1.6;
}
