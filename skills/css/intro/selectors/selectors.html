CSS Selectors

CSS selectors define which HTML elements a style rule should apply to. They are the foundation of targeting elements and applying styles in your web pages.

Common Selectors

  • Type Selector ‚Üí targets elements by tag name (e.g., p, h1)
  • Class Selector ‚Üí uses . before the class (e.g., .box)
  • ID Selector ‚Üí uses # before the ID (e.g., #header)
  • Universal Selector ‚Üí applies to all elements: *
  • Group Selector ‚Üí combine multiple selectors with commas: h1, p
  • Descendant Selector ‚Üí selects nested elements: div p

Example:

/* Select all paragraphs */
p {
  color: darkgreen;
}

/* Select elements with class 'note' */
.note {
  background-color: lightyellow;
}

/* Select element with id 'main-title' */
#main-title {
  color: navy;
  text-align: center;
}
