CSS Transitions & Animations

Introduction

CSS allows you to add motion and smooth changes using transitions and animations. These effects can make your interfaces more dynamic and engaging.

Transitions

  • Used for smooth changes between property values.
  • Common with hover effects, like button color or size.
  • Key property: transition: all 0.3s ease;

Animations

  • Enable more complex movements and sequences.
  • Use @keyframes to define animation steps.
  • Can loop, reverse, delay, and run indefinitely.

Example:

.box {
  transition: background-color 0.3s ease;
}

@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-20px); }
}
