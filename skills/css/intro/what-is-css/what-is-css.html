What is CSS?

CSS stands for Cascading Style Sheets. It describes how HTML elements should be displayed on the screen, paper, or in other media. While HTML structures your content, CSS makes it visually appealing.

üí° Why Use CSS?

Without CSS, all web pages would look plain and unstyled. CSS allows you to:

  • Change colors, fonts, and layouts
  • Create responsive designs for different screen sizes
  • Separate content (HTML) from design (CSS)

üîó Ways to Apply CSS to HTML

You can apply CSS to your HTML in three main ways:

1. Inline CSS

Use the style attribute inside an HTML tag:

<p style="color: blue;">This is a blue paragraph.</p>

2. Internal CSS

Define CSS rules inside a <style> tag in the <head> of your HTML document:

<!DOCTYPE html>
<html>
<head>
  <style>
    p {
      color: green;
    }
  </style>
</head>
<body>
  <p>This is a green paragraph.</p>
</body>
</html>

3. External CSS

Link to a separate CSS file using the <link> element:

<head>
  <link rel="stylesheet" href="styles.css">
</head>

And in styles.css file:

p {
  color: red;
}

üé® CSS Syntax Overview

A CSS rule consists of a selector and a declaration block:

selector {
  property: value;
}

Example:

h1 {
  color: purple;
  font-size: 24px;
}
