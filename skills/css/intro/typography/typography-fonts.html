CSS Typography & Fonts

Typography shapes the readability and aesthetic of your site. CSS lets you customize fonts, sizes, spacing, and more.

Key CSS Properties:

  • font-family: Defines the font type or stack
  • font-size: Controls text size
  • font-weight: Thickness of the text (e.g., normal, bold)
  • line-height: Vertical spacing between lines
  • letter-spacing: Space between characters
  • text-align: Alignment of text (left, right, center, justify)
  • font-style: Normal, italic, oblique

Example:

.typography-example {
  font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
  font-size: 1.1rem;
  font-weight: 600;
  line-height: 1.6;
  letter-spacing: 0.05em;
  text-align: justify;
  font-style: normal;
  color: #2c3e50;
}
