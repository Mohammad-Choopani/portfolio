What is HTML?

HTML stands for HyperText Markup Language. It is the fundamental building block of the Web. Every webpage you’ve ever visited was created using HTML — even this one!

In this lesson, you will learn:

  1. What HTML is and how it powers the web
  2. Basic structure of an HTML document
  3. How to write your first HTML page

Key Terms:

  • Tag: A keyword wrapped in angle brackets like <h1>.
  • Element: A complete set of start and end tags with content.
  • Attribute: Adds additional info to elements like href or src.

Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, HTML!</h1>
  </body>
</html>
