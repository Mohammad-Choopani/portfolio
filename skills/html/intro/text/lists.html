Lists & Tables

Lists and tables help organize content clearly. In this lesson, you'll learn how to use ordered and unordered lists, and how to create basic HTML tables.

Example List:

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Example Table:

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Ali</td>
    <td>30</td>
  </tr>
</table>
