Final Project: Build a Simple Webpage

Overview Requirements Steps Submission

Overview

In this final project, you'll create a simple, well-structured HTML webpage using all the concepts you have learned.

This project will demonstrate your understanding of HTML elements, semantic tags, media embedding, forms, and accessibility.

Requirements

  • Use semantic HTML tags like <header>, <nav>, <main>, <footer>.
  • Include a header with a title and navigation links.
  • Embed an image with descriptive alt text.
  • Add a short paragraph describing yourself or a fictional character.
  • Create a simple contact form with at least two input fields.
  • Make sure your page is accessible (e.g., use aria-labels where appropriate).

Steps

  1. Plan your webpage layout on paper or a design tool.
  2. Create an HTML file and structure it with semantic tags.
  3. Add your content, including text, images, and forms.
  4. Test your page in multiple browsers.
  5. Validate your HTML using online tools like the W3C Validator.

Submission

Upload your project files to your portfolio or GitHub repository. Share the link with your mentor or classmates for feedback.
