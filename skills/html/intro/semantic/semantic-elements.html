Semantic HTML Masterclass

<header>

Defines a page or section header.

<header>Welcome</header>

<nav>

Contains navigation links.

<nav><a href=\"#\">Home</a></nav>

<main>

Main content container.

<main>Content here</main>

<section>

Defines a document section.

<section>About us</section>

<article>

Independent self-contained content.

<article>Post content</article>

<aside>

Side information (e.g., sidebar).

<aside>Tip or note</aside>

<figure> + <figcaption>

Figure with caption (e.g., image).

<figure><img src=\"img.jpg\"><figcaption>This is a cat</figcaption></figure>

<mark>

Highlights part of the text.

<p>This is <mark>important</mark>.</p>

<time>

Represents a date/time.

<time datetime=\"2025-07-28\">July 28, 2025</time>

<details> + <summary>

Expandable content block.

<details><summary>More</summary>Hidden text here.</details>

<address>

Contact or location info.

<address>Mohammad, Windsor, Canada</address>

<abbr>

Defines an abbreviation with tooltip.

<abbr title=\"HyperText Markup Language\">HTML</abbr>
